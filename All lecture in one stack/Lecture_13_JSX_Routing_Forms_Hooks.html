Visual Aids (Board / Slides)

PDF diagrams text form

  • JSX compile: JSX → React.createElement → Virtual DOM diff → real DOM patch.
  • Routing: User clicks Link → URL → Router matches Route → component (no full reload).
  • Controlled flow: Input → onChange → state → UI. Uncontrolled: DOM stores value → ref se submit par read.
