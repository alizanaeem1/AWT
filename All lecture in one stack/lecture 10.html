Part A — 0 se 10 Minute

Angular Kyun Seekhein? 🤔

Jab app barhti hai, mushkilaat bhi barhti hain — state manage karna, code organize karna. Angular in sab ka hal deta hai!

📚
Framework vs Library
Angular poora framework hai — routing, DI, tooling sab built-in. React sirf library hai. Angular mein sab ready milta hai!
🏢
Enterprise ke liye Best
Banking apps, HR portals, admin dashboards — bade projects ke liye Angular ideal hai. Strong conventions se code maintainable rehta hai.
💪
TypeScript First
Type safety milti hai, errors compile time pe pakri jaati hain. Code professional aur error-free rehta hai!
🧩
Component-Based UI
Pura UI chhoti chhoti reusable components mein banta hai. Ek baar banao, hazaar jagah use karo!
💡
Ek Line SummaryAngular = Opinionated, scalable, TypeScript-first SPA framework. Ye yaad kar lo!

Part D — 45 se 65 Minute

Modules aur @NgModule 📦

Module Angular ka configuration map hai — batata hai kya compile karna hai, kya use karna hai!

app.module.ts 
@NgModule({
  declarations: [
    AppComponent,        // Mere apne components
    UserCardComponent    // Ye bhi mera component hai
  ],
  imports: [
    BrowserModule,       // Browser features
    FormsModule          // ngModel ke liye ZARURI!
  ],
  providers: [UserService],   // Services inject karo
  bootstrap: [AppComponent]   // App yahan se start hoti hai
})
export class AppModule {}
PropertyMatlabExample
declarationsMere apne components, pipes, directivesAppComponent
importsDoosre modules jo main use karta hoonFormsModule
providersServices jo inject karni hainUserService
bootstrapApp ka starting pointAppComponent
⚠️
Common Galtiyan!Component ko imports mein mat daalo — wo declarations mein jaata hai! ngModel use karo to FormsModule import ZARURI hai!
Part E — 65 se 90 Minute

Components aur @Component 🧩

Component ek reusable UI block hai. Selector + template + styles = Component ready!

user-card.component.ts 
@Component({
  selector: 'app-user-card',
  // HTML mein use:
  // <app-user-card></app-user-card>
  templateUrl: './user-card.html',
  styleUrls: ['./user-card.css']
})
export class UserCardComponent {
  userName = 'Ali Raza';
  role = 'Student';

  greet() {
    alert('Assalam o Alaikum!');
  }
}
🎯
Exam Tip!selector → HTML mein use hota hai → component ka template render hota hai. Ye link yaad rakho!
🔧
Shortcut — CLI se Generate!
Terminal mein likho:
ng generate component user-card
CLI khud 4 files bana deta hai — TS, HTML, CSS, spec!
📌
Yaad Rakhna!1 file logic ke liye (TS), 1 view ke liye (HTML). Separation of concerns!
Part F — 90 se 110 Minute

Data Binding — Angular ka Dil ❤️

Data binding component aur template ko connect karta hai. 4 types hain — ek ek samjho!

1️⃣
Interpolation — TS se HTML
Component ki value template mein dikhao. Double curly braces!
// Component:
userName = 'Ahmed';
// Template:
<h2>Hello, {{ userName }}</h2>
2️⃣
Property Binding — DOM set karo
HTML element ki property dynamically set karo. Square brackets!
// Component:
isDisabled = true;
// Template:
<button [disabled]="isDisabled">
  Submit
</button>
3️⃣
Event Binding — Action pakro
Click, input events pakro aur function chalao. Round brackets!
// Component:
save() { alert('Saved!'); }
// Template:
<button (click)="save()">
  Save
</button>
4️⃣
Two-Way Binding — Dono Taraf!
Input change → component update, component change → input update. "Banana in a box"!
// FormsModule import karo!
userName = '';
// Template:
<input [(ngModel)]="userName">
<p>{{ userName }}</p>

Quick Reference Table

TypeDirectionSyntaxUse Case
InterpolationTS → HTML{{ x }}Text dikhana
Property BindingTS → DOM[prop]="x"DOM property set
Event BindingDOM → TS(event)="fn()"User action handle
Two-WayTS ↔ HTML[(ngModel)]="x"Forms / inputs

Debugging Clinic 🐛

Sabse common errors aur unka hal — ye yaad rakho to kabhi phase nahi khao ge!

🔴
Error: "Can't bind to 'ngModel'"Matlab: FormsModule import nahi kiya. AppModule ke imports mein FormsModule add karo!
🔴
Component nazar nahi aa rahaMatlab: Selector spelling galat hai ya component declarations mein register nahi. Dono check karo!
🔴
Event fire nahi ho rahaMatlab: Wrong (click) placement ya method name mismatch. Template aur TS dono check karo!
💡
Pro TipBrowser DevTools Console kholo — error message khud batata hai kya problem hai. Hamesha console check karo pehle!
