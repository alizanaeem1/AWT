# 📚 Lecture Website – README

## 📌 Live Website

👉 https://lectures-awt.vercel.app/

---

## 📌 Overview

This project is a **modern, responsive dark-theme lecture website** where multiple lectures are organized in a clean and structured format. It is designed to help students understand concepts easily through **code examples, explanations, quizzes, and practice tasks**.

---

## 🎯 Purpose

The goal of this website is to:

* Explain complex topics in a **simple and clear way**
* Provide a **step-by-step learning experience**
* Combine **theory, practical examples, and quizzes** in one place

---

## 🧠 Features

### 🎨 UI/UX

* Modern dark theme design
* Fully responsive (mobile + desktop)
* Smooth animations (hover + fade effects)
* Sidebar navigation with active highlighting

### 📚 Lecture Structure

Each lecture includes:

* Hero section (title + introduction)
* Concept explanations
* Cards and grid layouts
* Tables for comparisons
* Alerts (important notes, warnings)
* Formatted code examples

### 🛠 Learning Sections

* Step-by-step explanations
* Real-world examples
* Debugging / common errors section
* Exam tips

### 🧪 Interactive Quiz

* 5 MCQs per lecture
* JavaScript-based interaction
* Instant feedback

### 🏠 Homework + Recap

* Practice tasks
* Mini assignments
* Key takeaways
