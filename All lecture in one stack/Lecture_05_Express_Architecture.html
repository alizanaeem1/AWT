Middleware function ko request object, response object, aur next middleware tak access — Express docs jaisi definition.

Routing

Endpoint → handler

GET, POST, PUT, DELETE — HTTP verbs. Route params: /user/:id → req.params.id. Maps analogy: alag URL → alag function.
